# Использование

https://www.figma.com/file/26Gj0HAuS32zVGjSjCqJRU/_matterjs_games_engine?node-id=0%3A1

Устанавливаем

```
npm install
```

Разработка

```
npm run dev
```

Сборка для продакшена

```
npm run build
```

Проверка сборки для продакшена

```
npm run serve
```
